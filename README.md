 # Data Visualization Project

This project demonstrates data visualization using Python.

## Tools & Libraries
- Python
- Pandas
- Matplotlib
- Seaborn

## Visualizations
- Category-wise Sales (Bar Chart)
- Monthly Sales Trend (Line Chart)
- Region-wise Sales Distribution (Pie Chart)

## Objective
To transform raw sales data into meaningful visual insights that support decision-making.
